Use the form below to configure and generate a QR code.

Required. Enter the full URL to the page you want to provide a link to, including the "https://"
Optional. Name / id sent to Google Analytics to record the specific promotion campaign so you can track the scans of this code. e.g. "spec_jazz_exhibit", "spec_kingspud". Only use on links to U of I Library websites!
Optional. Name or id for where you are putting the QR code so that you can track where they scanned it with Google Analytics e.g. "poster", "postcard", "sign". Only use on links to U of I Library websites! Use in conjunction with "Tracking campaign id".
Optional. Width of QR code to generate in pixels. Default is 300.
