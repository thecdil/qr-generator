Use the form below to configure and generate a QR code. Check the About page more tips and more information about Analytics usage.

Required. Enter the full URL to the page you want to provide a link to, including the "https://"
Optional. The name of the overall marketing campaign (that might have several different sources or mediums). Use a unique and identifiable name as this will be what we see in Analytics stats. E.g. "spec_jazz_2023", "spec_kingspud", "leganto_launch_2021". Only use on links to U of I Library websites!
Optional. The specific referrer that is sending the traffic to the page, identifying the specific item where you are putting the QR code. E.g. "lib-brochure2023", "fye-handout-1", "mill-sticker1". Only use on links to U of I Library websites! Use in conjunction with "Campaign" option.
Optional. The general marketing medium of the Source where you are putting the QR code. E.g. "email", "poster", "bookmark", "sign", or "display". Only use on links to U of I Library websites! Use in conjunction with "Campaign" option.
Optional. Width of QR code to generate in pixels. Default is 500. Set to a larger number for high res print quality.
